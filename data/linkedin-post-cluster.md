## GUIDE DE PUBLICATION

### Meilleur moment pour publier

- **Jour** : Mardi ou jeudi (engagement max sur LinkedIn FR)
- **Heure** : 7h30 - 8h30 du matin (les decideurs scrollent avant d'arriver au bureau)
- **Alternative** : Mardi 12h00-12h30 (pause dejeuner)
- Eviter : vendredi apres-midi, week-end, lundi matin

### Visuel a joindre

**Option ideale** : une photo reelle du cluster "La Creatrice" vu de face.
- Les 6 GPU visibles avec leurs LED allumees
- Un ecran en arriere-plan montrant un dashboard JARVIS (terminal avec des metriques)
- Eclairage ambiant, pas de flash direct — rendu "labo tech maison"
- Format carre (1080x1080) ou 4:5 (1080x1350) pour occupation maximale du feed

**Si pas de photo** : un schema technique propre sur fond sombre montrant les 4 noeuds (M1/M2/M3/OL1), les GPU de chaque noeud, et les fleches de communication. Style minimaliste, couleurs neon sur noir.

**Texte sur l'image** (optionnel, en gros) : "46 Go VRAM | 6 GPU | 0 cloud"

### Strategie du premier commentaire

Poster immediatement apres la publication un commentaire qui apporte de la valeur supplementaire. Exemples :

**Pour la Version A :**
> "Pour ceux que ca interesse, voici le detail du hardware :
> - RTX 3080 (10 Go) : inference lourde, modeles 7B+
> - RTX 2060 (6 Go) : taches secondaires, embeddings
> - 4x GTX 1660 Super (24 Go total) : batch processing parallele
> Le tout sous Ubuntu avec un orchestrateur Python maison. Je prepare un article technique detaille."

**Pour la Version B :**
> "Quelques chiffres pour les sceptiques :
> - Cloud GPU (A100) : ~2.50EUR/h soit ~1800EUR/mois en usage continu
> - Ce cluster : ~800EUR de matos d'occasion + 30EUR/mois d'electricite
> - ROI : 3 mois
> La souverainete des donnees, c'est en bonus."

### Strategie d'engagement (30 premieres minutes)

1. **T+0 min** : Publier le post + le premier commentaire dans la foulee
2. **T+5 min** : Envoyer le post en DM a 5-10 contacts pertinents avec un message personnalise : "Je viens de publier ca, ca pourrait t'interesser / j'aimerais ton avis"
3. **T+10 min** : Repondre a CHAQUE commentaire dans les 15 premieres minutes (meme un simple "merci" pousse l'algo)
4. **T+15 min** : Liker les commentaires des autres, poser des questions en retour pour relancer la discussion
5. **T+30 min** : Si un commentaire interessant emerge, y repondre avec un mini-paragraphe de valeur (pas juste "bien vu")
6. **Regle d'or** : Ne pas editer le post dans la premiere heure (LinkedIn penalise les edits rapides)
7. **Le lendemain** : Reposter le lien en story avec "Et vous, votre avis ?"

### Notes supplementaires

- **Version A** est recommandee pour un premier post (meilleur taux d'engagement, le storytelling performe 2x mieux que le contenu corporate sur LinkedIn FR)
- **Version B** peut etre postee 7-10 jours plus tard pour toucher un autre segment
- Alterner les deux versions permet de tester quel angle resonne le plus avec l'audience
- Penser a taguer 2-3 personnes pertinentes dans le premier commentaire (pas dans le post principal)
