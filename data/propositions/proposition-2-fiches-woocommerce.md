## Ce que je vous livre

### 1. Script de traitement par lot
- Vous déposez vos photos dans un dossier (Google Drive, Dropbox ou dossier local)
- Le script traite automatiquement chaque image : analyse, description, import WooCommerce
- Gestion des erreurs, journaux d'exécution, reprise en cas d'interruption

### 2. Workflow pour les ajouts quotidiens
- Chaque nouvelle photo ajoutée déclenche automatiquement la création de la fiche
- Vous n'avez qu'à prendre la photo et la déposer, le reste est automatique

### 3. Prompts spécialisés pour la seconde main
- Description de l'état : neuf, très bon état, bon état, traces d'usure, à restaurer
- Détection automatique des défauts visibles sur la photo
- Vocabulaire SEO adapté au marché de l'occasion
- Catégorisation et étiquetage automatiques

### 4. Garantie qualité
- **Les 10 premières fiches vous sont soumises pour validation** avant de traiter le reste
- Ajustements illimités sur les modèles de description jusqu'à votre entière satisfaction
- Mode brouillon : chaque fiche est créée en brouillon, vous validez en un clic
