## ÉTAPE 4 — Domaine franckdelmas.dev

### 1. Acheter le domaine

1. Aller sur **https://dash.cloudflare.com**
2. Menu gauche → **Domain Registration** → **Register Domain**
3. Chercher `franckdelmas.dev`
4. Acheter (paiement par CB déjà enregistrée)

### 2. Configurer le routage email

1. Dans Cloudflare, sélectionner le domaine `franckdelmas.dev`
2. Menu gauche → **Email** → **Email Routing**
3. **Create address** :
   - Adresse personnalisée : `franck`
   - Destination : `franckdelmas00@gmail.com`
4. Valider le mail de confirmation reçu sur Gmail

### 3. Configurer l'envoi depuis Gmail

1. Ouvrir **Gmail** → **Paramètres** (roue dentée) → **Voir tous les paramètres**
2. Onglet **Comptes et importation**
3. Section **Envoyer des e-mails en tant que** → **Ajouter une autre adresse e-mail**
4. Saisir : `franck@franckdelmas.dev`
5. Suivre les étapes de vérification
